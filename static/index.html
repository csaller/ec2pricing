EC2 Pricing

A static JSON API for AWS EC2 Linux On-Demand instance pricing, updated daily.

Live API

Individual instance pricing

https://www.ec2pricing.com/{region}/{family}/{size}.json

Examples:

# Get t3a.medium pricing in us-east-1
curl https://www.ec2pricing.com/us-east-1/t3a/medium.json

# Get c7g.2xlarge pricing in eu-west-1
curl https://www.ec2pricing.com/eu-west-1/c7g/2xlarge.json

Response format:

{
  "instanceType": "t3a.medium",
  "instanceFamily": "General purpose",
  "vCPU": "2",
  "memory": "4 GiB",
  "storage": "EBS only",
  "networkPerformance": "Up to 5 Gigabit",
  "operatingSystem": "Linux",
  "price": 0.0376
}

Family summaries

https://www.ec2pricing.com/{region}/{family}.json

Get all instances in a family, sorted by price.

Examples:

# Get all t3a instances in us-east-1
curl https://www.ec2pricing.com/us-east-1/t3a.json

# Get all m7g instances in eu-west-1
curl https://www.ec2pricing.com/eu-west-1/m7g.json

Response format:

{
  "family": "t3a",
  "region": "us-east-1",
  "instances": [
    {
      "instanceType": "t3a.nano",
      "instanceFamily": "General purpose",
      "vCPU": "2",
      "memory": "0.5 GiB",
      "storage": "EBS only",
      "networkPerformance": "Up to 5 Gigabit",
      "operatingSystem": "Linux",
      "price": 0.0047
    }
    // ... all instances in family, sorted by price
  ]
}
All price values are hourly On-Demand rates in USD.

Master index

# List all regions and instance types
curl https://www.ec2pricing.com/index.json

Returns a full manifest of every region and instance type:

{
  "updatedAt": "2026-02-17T06:00:00.000Z",
  "regions": {
    "us-east-1": {
      "instanceTypes": ["c5.xlarge", "m5.large", "t3.micro", "..."]
    }
  }
}

Usage Examples

Examples of fetching EC2 pricing data from the API in different programming languages:

Fetch data from a single instance:

async function getInstanceData(region, family, size) {
  const url = `https://www.ec2pricing.com/${region}/${family}/${size}.json`
  const response = await fetch(url)
  const data = await response.json()
  return data
}

const instanceData = await getInstanceData('us-east-1', 't3a', 'medium')
console.log(`Instance: ${instanceData.instanceType}`)
console.log(`Price: $${instanceData.price}/hour`)
console.log(`vCPU: ${instanceData.vCPU}, Memory: ${instance.memory}`)

Fetch data from all instances on a family:

async function getFamilyData(region, family) {
  const url = `https://www.ec2pricing.com/${region}/${family}.json`
  const response = await fetch(url)
  const data = await response.json()
  return data
}

const familyData = await getFamilyData('us-east-1', 't3a')
console.log(`Family: ${familyData.family}`)
console.log(`Instance count: ${familyData.instances.length}`)

Fetch data from a single instance:

import requests

def get_instance_data(region, family, size):
    url = f"https://www.ec2pricing.com/{region}/{family}/{size}.json"
    response = requests.get(url)
    response.raise_for_status()
    return response.json()

instance = get_instance_data('us-east-1', 't3a', 'medium')
print(f"Instance: {instance['instanceType']}")
print(f"Price: ${instance['price']}/hour")
print(f"vCPU: {instance['vCPU']}, Memory: {instance['memory']}")

Fetch data from all instances on a family:

import requests

def get_family_data(region, family):
    url = f"https://www.ec2pricing.com/{region}/{family}.json"
    response = requests.get(url)
    response.raise_for_status()
    return response.json()

family_data = get_family_data('us-east-1', 't3a')
print(f"Family: {family_data['family']}")
print(f"Instance count: {len(family_data['instances'])}")

Data source

Pricing data is fetched directly from the official AWS EC2 Pricing API.

Filtering:

  • Only Linux On-Demand pricing is included
  • Standard BoxUsage instances only (no Reserved, Savings Plans, Dedicated, or Spot)
  • Base Linux without pre-installed software (SQL, etc.)
  • No Windows, RHEL, SUSE, or other operating systems
